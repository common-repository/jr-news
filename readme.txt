== Description ==

This plugin allows you to display the latest news updates directly on your blog - News from specific countries, technology news, political news, sports news and even odd news!

Simply enable the plugin through your admin panel, edit the JR News settings in the Settings --> JR News menu and enable the widget through the Appearance --> Widgets menu! News will now be displayed on your blog!

== Installation ==

1. Upload `jr_news.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR News Settings in your Administration Panel
4. Enable the Widget through the Appearance --> Widgets menu.
